# Calculadora de Propinas

Este programa pide el total de la cuenta, el porcentaje de propina que desean dejar y el número de personas. El mismo debe devolver cuánto debe pagar cada uno.

* Conceptos: Entrada de datos (input), conversión de tipos (float, int) y operaciones aritméticas.
