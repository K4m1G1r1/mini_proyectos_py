# Conversor de Unidades (Fahrenheit a Celsius)

Programa que reciba una temperatura en grados Fahrenheit y la transforma a grados Celsius.

* Conceptos: Fórmulas matemáticas básicas y concatenación de strings.
* Fórmula: $C = (F - 32) \cdot 5/9$ o $C = (F - 32) / 1.8$
