## Explicación Rápida

### Mini Proyecto 01: Calculadora de Propinas

Este programa pide el total de la cuenta, el porcentaje de propina que desean dejar y el número de personas. El mismo debe devolver cuánto debe pagar cada uno.

* Conceptos: Entrada de datos (input), conversión de tipos (float, int) y operaciones aritméticas.

### Mini Proyecto 02: Conversor de Unidades (Fahrenheit a Celsius)

Programa que reciba una temperatura en grados Fahrenheit y la transforma a grados Celsius.

* Conceptos: Fórmulas matemáticas básicas y concatenación de strings.
* Fórmula: $C = (F - 32) \cdot 5/9$ o $C = (F - 32) / 1.8$

### Mini Proyecto 03: El Juego de Adivinar el Número

La computadora elige un número al azar entre 1 y 50, y el usuario tiene que adivinarlo. El programa debe decir si el número es "más alto" o "más bajo" hasta que el usuario acierte.

* Conceptos: Importar módulos (import random), bucles while y comparaciones.

### Mini Proyecto 04: Contador de Palabras y Caracteres

El usuario ingresa un párrafo o una oración, y el programa le dice cuántas palabras y cuántas letras (sin espacios) tiene el texto.

* Conceptos: Métodos de strings (.split(), .replace()) y la función len().
